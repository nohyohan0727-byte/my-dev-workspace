# My Dev Workspace

두 컴퓨터 간 개발 형상관리를 위한 작업 저장소입니다.

## 구조

```
my-dev-workspace/
├── projects/     # 프로젝트별 폴더
└── notes/        # 개발 메모 및 참고사항
```

## 자주 쓰는 Git 명령어

### 작업 시작할 때 (다른 컴퓨터에서 최신 내용 가져오기)
```bash
git pull
```

### 작업 저장할 때
```bash
git add .
git commit -m "작업 내용 설명"
git push
```

### 상태 확인
```bash
git status
git log --oneline
```
