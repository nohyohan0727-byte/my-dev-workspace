## index.html 주요 변경

### 타이틀/카피
- Hero h1: `반복 서류 업무에서 벗어나 / 본업에 집중하세요` → `대표님 대신 서류 쓰는 / AI 비서를 채용하세요`
- Hero 부제목: `업무 자동화 시스템으로` → `AI 비서가 대신 처리해`
- "AI 자동화 시스템" → "AI 비서" 전체 치환

### UX 개선
- `자동화 사례 보기` 버튼 앵커 버그 수정: `href="#services"` → `href="#cases"` (automation-examples-section에 id 추가)
- 로고 `<div>` → `<a href="index.html">` (클릭 시 홈 이동)
- 네비게이션 `position: fixed` → `position: sticky` (이벤트 배너 가림 문제 해결)
- 히어로 `padding-top: 140px` → `80px` (fixed 보정 패딩 제거)

### 신규 요소
1. **Hero 이벤트 버튼** (빨간 계열)
   - `🔥 3월 한정` 뱃지 + "무료 진단 선착순 10명 · 지금 신청하면 당일 연락"
   - `href="index2.html"` 연결
   - CSS: border 밝기 깜빡임 애니메이션 (`hero-event-glow`)

2. **브릿지 배너** (자동화 사례 섹션 뒤, 서비스 섹션 앞)
   - "어떤 자동화부터 시작해야 할지 아직 감이 안 잡히신다면"
   - 보라색 그라데이션 배경, `href="index2.html"` 연결

3. **플로팅 CTA 버튼** (우측 하단 고정)
   - 300px 스크롤 후 부드럽게 등장 (`opacity: 0 → 1`, JS)
   - `href="index2.html"` 연결

4. **상담 폼 하단 직통 연락처**
   - 제이케이네트웍스 대표 노진광
   - `<a href="tel:010-3127-4528">📞 010-3127-4528</a>` (모바일 탭 전화 연결)

## index2.html (무료상담 전환 랜딩 v1)

### 설계 원칙
- 기능 설명 최소화, "대표 시간 손실" 직접 자극
- 무료 진단 강조 반복, CTA 버튼 5회 삽입
- 긴박감 + 영업 불안 제거 문구 균형

### 섹션 구성
1. **Hero**: "서류 때문에 야근하십니까?" + 첫 번째 CTA
2. **체크리스트**: 해당 상황 5가지 확인
3. **손실 비용**: 인건비·실수·대표시간 3카드
4. **차별점**: 범용 AI 툴과 다른 점, Before/After 속도 비교
5. **직원교육 대체**: BEFORE/AFTER + 실제 대화 시나리오 (신입→AI비서 질문)
6. **LIVE DEMO**: RAG 시스템 미리보기 카드, register.html 연결
7. **고객 후기**: 도입 전/후 3개 (제조업·IT·유통업)
8. **상담 무료 이유**: 신뢰 3포인트
9. **긴박감**: 잔여 슬롯 진행바 (JS 동적 계산)
10. **Final CTA**
11. **상담 폼**: Netlify 연동 + 직통 연락처

### 카피 차별화 원칙
- CTA 버튼마다 다른 문구 (중복 없음)
- 섹션 레이블 한국어 통일
- "10분이면 방향이 정해집니다" 중복 제거
- "무료" 과도 사용 → 자연스럽게 분산

## index3.html (전환 최적화 고도화 v2)

### index2 대비 업그레이드

| 요소 | index2 | index3 |
|------|--------|--------|
| Hero 심리 자극 | 야근 질문 | "대표 시간 = 가장 비싼 자원" 추가 |
| 수치 | 없음 | 월 40h+ / 90% 단축 / 10분 (카운트업 JS) |
| 손실 카드 | 3개 | 4개 (성장 집중 불가 추가) |
| Before/After | 텍스트 | 그리드 + "최대 90%↓" 배지 |
| 보증 문구 | 분산 | Final에 4알약 집중 (비용·부담·강요·10분) |
| 폼 항목 | 4개 | 5개 (신입교육 추가) |
| 배경 | 단색 | 격자 패턴 오버레이 |

### 기술 특이사항
- `IntersectionObserver`로 손실 수치 카운트업 (화면 진입 시 발동)
- CSS 변수 기반 색상 시스템 (`--bg`, `--bg2`~`--bg4`, 의미 단위 구분)
- `@keyframes quota-pulse` 진행바 애니메이션
